# Practica-RAB
Экзаменатор: Система подготовки к экзаменам
Описание
"Экзаменатор" — это консольное приложение, предназначенное для помощи студентам в подготовке к экзаменам. Система позволяет создавать, проходить тесты, анализировать результаты и отслеживать прогресс. Приложение разработано в рамках учебной практики в Пензенском колледже информационных и промышленных технологий (ИТ-колледж).

Основные возможности
Авторизация пользователей (студент/преподаватель)
Прохождение тестирования по выбранным предметам и темам
Просмотр статистики (личной для студента, общей для преподавателя)
Управление тестами и вопросами (для преподавателей)
Результаты тестирования в реальном времени с подсчётом правильных ответов и процента успеха
Системные требования
Операционная система: Windows 10/11 или совместимые дистрибутивы Linux
Среда выполнения: .NET 6.0 или выше
Минимальная конфигурация:
Процессор: 1 ГГц
ОЗУ: 512 МБ
Свободное место на диске: 50 МБ
Установка и запуск
Убедитесь, что установлена среда выполнения .NET 6.0 или выше
Склонируйте или скачайте проект
Перейдите в директорию проекта
Выполните команды:
dotnet build
dotnet run
Использование
Для студентов:
Авторизуйтесь в системе
Выберите "Пройти тест"
Выберите предмет и тему
Отвечайте на вопросы, вводя номер варианта ответа
По окончании просмотрите результат
Для анализа прогресса выберите "Моя статистика"
Для преподавателей:
Авторизуйтесь под учётной записью преподавателя
Доступны функции добавления/удаления тем и вопросов
Просмотр общей статистики по всем студентам
Технические детали
· Язык программирования: C# · Архитектура: объектно-ориентированное программирование (ООП) · Интерфейс: текстовый консольный (CLI) · Хранение данных: файловая система (JSON/CSV) · Среда разработки: Visual Studio Code / Visual Studio
