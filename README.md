# Экзаменатор: Система подготовки к экзаменам

## Описание
"Экзаменатор" — это консольное приложение, предназначенное для помощи студентам в подготовке к экзаменам. Система позволяет создавать, проходить тесты, анализировать результаты и отслеживать прогресс. Приложение разработано в рамках учебной практики в Пензенском колледже информационных и промышленных технологий (ИТ-колледж).

## Основные возможности
- **Авторизация пользователей** (студент/преподаватель)
- **Прохождение тестирования** по выбранным предметам и темам
- **Просмотр статистики** (личной для студента, общей для преподавателя)
- **Управление тестами и вопросами** (для преподавателей)
- **Результаты тестирования в реальном времени** с подсчётом правильных ответов и процента успеха

## Системные требования
- **Операционная система:** Windows 10/11 или совместимые дистрибутивы Linux
- **Среда выполнения:** Android Studio-2025 г.
- **Минимальная конфигурация:**
  - Процессор: 1 ГГц
  - ОЗУ: 512 МБ
  - Свободное место на диске: 50 МБ

## Установка и запуск
1. Убедитесь, что установлена среда выполнения Android Studio-2025 г.
2. Склонируйте или скачайте проект
3. Перейдите в директорию проекта
4. Выполните команды:
   ```bash
   dotnet build
   dotnet run

# Использование

## Для студентов:

1. Авторизуйтесь в системе
2. Выберите тему
3. Отвечайте на вопросы, выбирая номер варианта ответа
4. По окончании просмотрите результат
5. Для анализа прогресса выберите "Просмотр статистики"

## Для преподавателей:

1. Авторизуйтесь под учётной записью преподавателя
2. Доступны функции добавления/удаления тем и вопросов
3. Просмотр общей статистики по всем студентам
